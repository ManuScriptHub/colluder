# Colluder - The Elm Frontend

This is the editor for Colluder.  It is essentially a Tracker-style editor for
composing songs.

## Features

- [x] Arbitrarily many tracks
- [x] Modifiable BPM
- [x] MIDI SoundFonts for instruments
- [ ] Choose the instrument for a given track
